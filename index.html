Bienvenidos a nuestra pagina GitHub.

Como istalar Cocos Creator. A continuacion el lick para descargar Cocos Creator Cocos Creator, Selecciona el Cocos Creator dependiendo de tu sistema sistema operativo Mac o Windows, (CocosCreator_v1.3.3_mac o CocosCreator_v1.3.3_win)

Para descargar ------->Python

Italacion.

Una vez descargado Cocos Creator ejecutas el archivo (.exe) se ejecutara como administrador.

En seguida aparecera una venta selecionas el idioma con el que quieres que se configure Cocos Creator.

Despues se abrira una ventana donde le diras donde quieres que se guarde, aceptas la licencia con un click en la casilla y le das click en istall

Enseguida se abriara una ventana donde te pide Visual Estudio para que copiles el programa que de desee crear una vez dado click se empezara a descargar Visual estudio 2005 y automaticamente se istalara Cocos Creator

Por ultimo te da dos opciones correr el programa o terminar

Listo se a istalado Cocos Creator
